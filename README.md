# StarsBot

## Installation

Add this line to your application's Gemfile:

add a `.env` file in the root of the folder with the following:

```
SLACK_API_TOKEN=your-slack-token
CHECK_EVERY=10
NOTIFY_WITH=10 
```

## run the application

```ruby
gem install foreman
bundle install
foreman start
```
